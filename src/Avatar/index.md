## 属性

| 属性 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| size | 'x-small' &verbar; 'small' &verbar; 'medium' &verbar; 'large' &verbar; number | 否 | 'medium' | x-small(80 x 80); small(88 x 88)；medium(104 x 104); large(120 x 120) |
| src | string | 否 | - | 头像地址，默认为灰色的内置图片 |
| className | string | 否 | - | 类名 |
| style | string | 否 | - | 样式 |
